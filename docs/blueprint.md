# **App Name**: Xelaris - Software Testing Experts

## Core Features:

- Homepage Hero Video: A full-screen video background on the homepage featuring software testing or developer-related visuals.
- Sticky Navigation: A persistent navigation bar that remains at the top of the screen as the user scrolls. Includes links to Home, Services, About Us, Case Studies, Careers, Blog, and Contact.
- Service Cards with Hover Effects: Interactive cards displaying Xelaris's services: Manual Testing, Automation Testing, API Testing, Performance Testing, Mobile App Testing.
- Animated Statistics: Display animated counters for key metrics like test cases, clients, and bugs found.
- Case Study Carousel: Carousel displaying customer feedback and successful case studies, creating a positive impression.
- Careers Section with Application: Job postings section including titles such as QA Engineer, Automation Engineer. A way to respond and express interest in available jobs.
- Contact Form: Contact page with an embedded form, map, and company details.

## Style Guidelines:

- Primary color: Deep blue (#224293) to convey professionalism and trust.
- Background color: Light grey (#F0F4F8), providing a clean and modern feel.
- Accent color: Neon green (#39FF14) as a vibrant highlight color.
- Font: 'Inter', a sans-serif for a clean and modern look in both headings and body text.
- Use a clean, tech-focused layout similar to codoid.com, ensuring a professional appearance.
- Implement smooth animations for on-scroll reveals and hover effects, engaging users as they navigate the site.
- Use subtle parallax scrolling effects to add depth and visual interest.
